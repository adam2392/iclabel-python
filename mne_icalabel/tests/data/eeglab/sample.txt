The file 'sample.set' was obtained by using the sample EEGLAB dataset.
The EOG channels have been dropped and the dataset has been cropped between 0
and 10 seconds.

.. code-block:: Matlab

    % Load dataset
    file_dataset = '/eeglab2022.0/sample_data/eeglab_data.set';
    EEG = pop_loadset(file_dataset);
    EEG = eeg_checkset(EEG);

    % Drop non-EEG channel and crop dataset
    idx = eeg_chaninds(EEG, {'EOG1', 'EOG2'})
    EEG = pop_select(EEG, 'nochannel', idx, 'time', [0, 10])

    % Run ICA
    EEG = pop_runica(EEG, 'icatype', 'runica', 'extended', 1, 'interrupt', 'off');

    % Save
    pop_saveset(EEG, 'filename', 'sample.set', 'savemode', 'onefile')

.. code-block:: Python

    from mne.io import read_raw
